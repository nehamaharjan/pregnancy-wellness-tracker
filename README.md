# Pregnancy Wellness Tracker App
A web-based pregnancy support and information system developed using PHP and MySQL as part of a BIM academic project.

# Project Overview
This system is designed to help users track pregnancy-related information, manage basic health guidance, and provide an organized platform for pregnancy wellness support.
It includes both user and admin modules with proper authentication and structured backend logic.

# Features

User Side
* User registration and login system
* Dashboard for pregnancy-related information
* Access to system features and guidance
* Profile management

Admin Side
* Admin login system
* Manage users and data
* Control system content

Technologies Used
* PHP (Backend)
* MySQL (Database)
* HTML (Frontend structure)
* CSS (Styling)
* JavaScript (Basic interactivity)

# Security Note
SMTP credentials and sensitive data were removed from the project before uploading to GitHub for security reasons.
